1. **A Methodology for Selecting and Composing Runtime Architecture Patterns for Production LLM Agents**

**Publish Date:**
2026-05-20

**一句话总结:**
提出随机确定性边界(SDB)概念和六种运行时模式(分层委托、散集+ Saga、事件驱动排序、共享状态机、监管+门禁、人在回路)以组合生产级LLM代理的协调、状态和控制，并提供选型方法论、故障诊断和重放分歧分析，附带一个合同续签代理的参考实现。

**Link:**
https://arxiv.org/abs/2605.20173

2. **How Far Are We From True Auto-Research?**

**Publish Date:**
2026-05-20

**一句话总结:**
该研究通过ResearchArena让三种AI代理自主完成科研全流程并生成117篇论文，经带制品评审发现实验严谨性是主要瓶颈，存在结果捏造等严重问题，无一篇论文达到顶会接收标准，揭示了当前自动科研系统与真正自主研究间的显著差距。

**Link:**
https://arxiv.org/abs/2605.19156

3. **Library Drift: Diagnosing and Fixing a Silent Failure Mode in Self-Evolving LLM Skill Libraries**

**Publish Date:**
2026-05-20

**一句话总结:**
该研究发现自演进LLM技能库中存在‘库漂移’失效模式，即无结果驱动的技能累积导致性能停滞，提出基于结果退役、容量限制和元技能先验的治理方案，结合可追溯诊断，将MBPP+ hard-100的pass@1从0.258提升至0.584，为自演进代理的技能维护提供了具体修复方法。

**Link:**
https://arxiv.org/abs/2605.19576

4. **Embedding by Elicitation: Dynamic Representations for Bayesian Optimization of System Prompts**

**Publish Date:**
2026-05-20

**Link:**
https://arxiv.org/abs/2605.19093

8. **DecisionBench: A Benchmark for Emergent Delegation in Long-Horizon Agentic Workflows**
